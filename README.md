# Spice My Rice – Mobile AR Android App

Mobile Augmented Reality app for visualization of food decoration patterns.

---

## Overview

This repository contains the **Mobile AR prototype** android application developed for the *Spice My Rice* research project. These repository contains the Android studio project files that implements the application that fetches and presents the lens in the repository PowderAR_Lens. The system explores how AR visualization can support **food decoration tasks** by projecting decorative patterns directly onto the food surface.

The prototype allows users to preview and manipulate decorative patterns in AR before executing them manually using traditional ingredients such as cinnamon or whipped cream. The goal is to support both **conceptualization and execution** of decorative designs by providing spatial visual guidance.

This prototype corresponds to the **mobile AR implementation described in the paper**.

## Features

- AR pattern visualization through marker tracking
- 2D UI interaction for pattern height, opacity and design
